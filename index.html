Objetivos del OVI

Objetivo general: Identificar, clasificar y utilizar herramientas tales como los sistemas de control de versiones y editores de código par la web, además, crear un guion y una maquetación para el diseño de un OVI en formato HTML

Objetivo específico 1: Identificar algunos referentes de diseño fundamentales y que han hecho eco a través del tiempo en el desarrollo de la web.

Objetivo específico 2: Identificar las características esenciales de los sistemas de control de versiones para la web con el fin de optar por los apropiados.

Objetivo específico 3: Identificar y apropiar editores de páginas web para facilitar la creación y edición de documentos HTML y CSS3.
